# Structural Prediction and Domain Analysis of BRCA1 Using AlphaFold

## Overview
This project focuses on the structural analysis of the BRCA1 protein using the AlphaFold Protein Structure Database. The study includes protein structure visualization, pLDDT confidence analysis, Predicted Aligned Error (PAE) interpretation, and domain organization analysis.

---

## Protein Information

| Parameter | Details |
|---|---|
| Protein | BRCA1 |
| Organism | Homo sapiens |
| UniProt ID | P38398 |

---

## Tools Used
- AlphaFold Protein Structure Database
- UniProt
- Protein Data Bank (PDB)

---

## Workflow
1. Retrieve BRCA1 structure from AlphaFold  
2. Visualize the predicted 3D structure  
3. Analyze pLDDT confidence scores  
4. Examine PAE plots  
5. Study domain organization and flexible regions  

---

## Key Findings
- BRCA1 contains both structured and disordered regions
- High-confidence domains were identified using pLDDT scores
- PAE analysis revealed flexible linker regions
- Domain analysis showed modular structural organization
